# ARNavigation 

Naver Maps를 활용한 ARNavigation  

## 개요

- Naver SDK iOS 연동
- Naver Direction API를 활용하여 일반 길찾기 기능 구현
- ARKit을 활용하 다양한 기능 추가


## 아이디어

### 내가 다녀온 곳 AR 스태츄 남기기  

내가 다녀 간곳에 ARKit을 통해 나만의 글자, 캐릭터, AR모델을 렌더링 학 기록한다. 지인들과 공유할 수 있어 커뮤니티를 형성 할 수 있다.
나의 발자취를 또는 친구의 발자취를 AR을 통해 느껴보자..
